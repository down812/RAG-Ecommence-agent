# RAG-E-commerce 智能电商平台技术文档

## 一、项目概述

RAG-E-commerce 是一个基于**检索增强生成(RAG)**技术的智能电商平台，采用前后端分离架构：

- **后端**：Spring Boot 3.2.0 + Spring AI 1.0.0-M6
- **前端**：Android 原生应用 (Jetpack Compose + Kotlin)
- **核心技术**：向量数据库检索 + 大语言模型对话 + 电商核心功能

## 四、RAG架构设计

### 4.1 RAG技术架构

```
用户输入 → ChatService.analyse()
         ↓
   [Spring AI ChatClient]
         ↓
   [DatabaseChatMemory] → 从MySQL读取历史对话
         ↓
   [ProductTool] → AI调用商品搜索工具
         ↓
   [VectorKnowledgeService] → ES向量库检索
         ↓
   [LLM推理] → 结合检索结果生成回答
         ↓
   流式返回AIChatResponse
```

### 4.2 RAG核心组件

#### ChatServiceImpl

核心对话处理服务，支持流式SSE输出：

- 解析AI返回的**结构化输出**（推荐/搜索/图片识别）
- 调用`ProductTool`获取真实商品数据
- 协调各组件完成RAG流程

#### VectorKnowledgeServiceImpl

向量知识库服务：

- `save()` - 保存文档到ES向量库
- 支持PDF、JSON、DOCX、TXT等格式
- 使用`TokenTextSplitter`进行文档分词
- `similaritySearch()` - 向量相似度搜索

#### ProductTool

AI可调用商品查询工具（@Tool注解）：

- `getProductByIds()` - 根据ID查询商品
- `searchProducts()` - 多条件搜索商品
- `getHotProducts()` - 获取热门商品

#### DatabaseChatMemory

数据库对话记忆实现：

- 实现`ChatMemory`接口
- 对话历史存储到MySQL的`log`表
- 支持中断标记（stop signal）

### 4.3 AI响应结构

AI返回三种结构化响应类型：

```java
public class AIChatResponse {
    private String type;  // recommendation / search_result / image_search

    // recommendation类型
    private List<AIRecommendationVO> recommendations;

    // search_result类型
    private List<AISearchResultVO> searchResults;

    // image_search类型
    private List<ImageSearchResult> imageResults;
}
```

## 七、技术亮点

1. **RAG增强对话**：结合Elasticsearch向量检索与商品数据库真实数据
2. **结构化输出**：AI返回标准化的商品推荐/搜索结果，便于前端渲染
3. **流式响应**：SSE实现实时流式对话体验
4. **AI Tools**：通过@Tool注解让AI调用商品搜索功能
5. **数据库记忆**：对话历史持久化到MySQL，支持多会话管理
6. **多格式文档支持**：PDF、JSON、DOCX等文档向量化入库
7. **前后端分离**：Android原生应用提供流畅用户体验
